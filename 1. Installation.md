# Ollama

Start building with open models. Ollama is a platform that helps you download and run AI models locally on your laptop.

### Why Ollama

Using Ollama is ideal because:

1. you can run models locally
2. no API billing anxiety
3. no cloud account setup
4. works even with limited internet
5. helps you understand AI practically instead of treating it like “magic”

## “AI is not only cloud. I can run it myself.”

Technically:

- Ollama itself is not the AI model
- Ollama is the runtime/host/platform
- Models like:
  - Meta's Llama
  - Microsoft Phi
  - Google Gemma
  are separate downloadable LLMs.

## Download

### macOS

```shell
curl -fsSL https://ollama.com/install.sh | sh
```

or [download manually](https://ollama.com/download/Ollama.dmg)

### Windows

```shell
irm https://ollama.com/install.ps1 | iex
```

or [download manually](https://ollama.com/download/OllamaSetup.exe)

### Linux

```shell
curl -fsSL https://ollama.com/install.sh | sh
```
